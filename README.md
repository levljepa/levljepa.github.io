# LeVLJEPA

This is the repository that contains source code for the [LeVLJEPA website](https://levljepa.github.io).

This project page is built on top of the [LeWorldModel](https://le-wm.github.io) website template by [Lucas Maes](https://x.com/lucasmaes_) et al. We thank them for making their code publicly available.
